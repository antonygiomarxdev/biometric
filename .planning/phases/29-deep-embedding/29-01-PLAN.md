# Plan 29-01: Embedding Pipeline + Full Legacy Cleanup

**Phase:** 29 (Deep Fingerprint Embedding)
**Goal:** Replace the entire MCC/minutiae pipeline with AFR-Net embedding. No coexistence.
**Estimated tasks:** 12
**Files:** ~9 new, ~25 modified, ~25 deleted

## Scope

This plan does two things atomically:

1. **Integrate** AFR-Net + U-Net embedding as the sole matching pipeline.
2. **Delete** all MCC/Bozorth3/minutiae/ridge-graph legacy code, configs, migrations, and Qdrant collections.

No dual-write, no feature flags, no gradual migration. The classical pipeline is gone when this plan completes.

## Architecture (final state)

```
                     FastAPI
                        │
     POST /fingerprints/{id}/captures    POST /matching/search
                  (existing)                (replaces /matching/search)
                        │                        │
                        ▼                        ▼
              FingerprintEnrollmentService   EmbeddingService
                        │                        │
        ┌───────────────┼────────────┐          │
        ▼               ▼            ▼          ▼
   PG: Capture     MinIO: image   Qdrant:   embed(512-D)
   (real UUID)    (captures/uuid)  fingerprint_embeddings
                                    payload: {person_id, capture_id}
                        │                        │
                        └──── shared ────────────┘
                              EmbeddingService
```

### T6: Rewrite FingerprintEnrollmentService

Replace `src/services/fingerprint_enrollment_service.py`:
- Inject `EmbeddingService` (not `MccMatchingService`)
- `create_capture(fingerprint_id, image_bytes, ...)`:
  1. Create `FingerprintCapture` row in PG with `algorithm_version="afrnet-v1"`
  2. Upload **original image** to MinIO at `captures/{capture_id}.png` (not skeleton)
  3. Call `embedding_service.embed(image_bytes)` → 512-D vector
  4. Call `embedding_service.qdrant.upsert(capture_id, vec, payload={person_id, capture_id})`
  5. Return `(capture, embedding_id)`
- Remove all minutiae extraction, skeletonization, pair indexing
- Remove `_hash_minutia` method

**Files:** 1 modified.
