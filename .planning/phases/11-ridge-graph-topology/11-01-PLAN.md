<objective>
Implement the Ridge Graph Extractor to map the biological fingerprint network (the "mycelium").

Purpose:
Instead of imposing arbitrary mathematical shapes (like Delaunay triangles), we will trace the actual fingerprint ridges. Using the `sknw` (Skeleton Network) library, we can automatically parse a 1-pixel thick skeleton image into a formal Graph where bifurcations/terminations are Nodes, and the ridges connecting them are Edges. This captures the true elastic-resistant topology.

Output: `RidgeGraph` domain entity and `RidgeGraphExtractor` processing step.
</objective>
