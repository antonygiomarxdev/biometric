    - `handleMouseMove`: In "move" mode, update position of selected minutia.
    - `handleSave`: Returns the current minutiae array for submission.
    - Redraw canvas after every mutation (image + updated minutiae overlay).

    Keep all existing animation logic intact (the animated minute draw on load). Add editing as opt-in via the new `mode` state.
