<objective>
Implement a bidirectional anonymizer `DataMasker` to replace sensitive PII (like names) with tokens (e.g., `[PERSON_1]`) before sending data to generative AI models, and to reverse this tokenization upon receiving responses.

Purpose: Prevent leaking raw forensic subject data to LLMs per ADR 007.
Output: Bi-directional data masking service.
</objective>
