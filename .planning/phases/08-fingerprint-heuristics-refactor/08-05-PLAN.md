<objective>
Transform the uniform pipeline into a high-performance, async, GPU-accelerated engine capable of handling partial/latent crime-scene prints.

Purpose: 
1. **Performance:** CPU execution blocks the GIL and takes ~10s per image. We must use `cupy` to offload Gabor filters and orientation fields to the RTX 4070 GPU, and use `asyncio` to prevent blocking the FastAPI event loop.
2. **Latent Matching:** Global normalization (Center of Mass + PCA) assumes a full finger. A crime-scene fragment has no true center. We must implement Delaunay Triangulation (Minutiae Triplets) so the matching relies on local geometry (angles/distances between 3 adjacent minutiae), which remains identical regardless of print rotation, missing borders, or translation.
Output: A production-ready, async, GPU-powered Latent AFIS pipeline.
</objective>
